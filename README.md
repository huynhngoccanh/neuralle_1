# Jordan Michaelides personal website

## Setup
* Clone the repo.
* Add environment variables:
  * `MEETUP_API_KEY`
  * `REDIS_URL`
  * `S3_KEY`
  * `S3_SECRET`
  * `S3_BUCKET`
  * `S3_REGION`
  * `MAILGUN_SMTP_PASSWORD`
  * `MAILGUN_SMTP_LOGIN`
  * `MAILGUN_SMTP_SERVER`
  * `MAILGUN_SMTP_PORT`
* Bundle and go!
